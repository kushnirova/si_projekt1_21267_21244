# Problem doboru projektów

Algorytm ma dobierać optymalny zestaw projektów, każdy z których ma czas wykonania, koszty oraz zysk, a wybrany zestaw projektów musi spełniać następujące wymagania:
- łączny koszt projektów nie przekracza kwoty X
- łączny czas wykonania wszystkich projektów mieści się w 12 miesiącach
- łączny zysk jest większy od Y
